## 1. 프로젝트 개요
### 프로젝트 소개 
1. OUR POS : 햄버거 프랜차이즈 가맹점들에게 배포되는 주문 및 배송 관리 솔루션 <br>
2. 구현 기능 <br/>
   2-1) 메뉴-사장님 : 메뉴 추가, 메뉴 수정, 메뉴 삭제, 메뉴 확인 <br/>
   2-2) 주문-사장님 : 주문 확인, 주문 승인, 주문 거절, 서빙 완료, 결제 완료 <br/>
   2-3) 장바구니-손님 : 장바구니 추가, 삭제  <br/>
   2-4) 주문-손님 : 메뉴 주문, 주문 금액 확인
